## Installation

```sh
mkdir -p ~/.config/ulauncher/user-themes
git clone https://github.com/GiorgioReale/Ulouncher-Essential-Light-Theme.git \
  ~/.config/ulauncher/user-themes/Essential-Light-Theme
```
